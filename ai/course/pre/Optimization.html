# 1、证明方法和相关记法

# 1.1、证明方法

真值表 ——> 德摩根律

  • (AB)' = A'+B'
  • (A+B)' = A'B'

命题等价于其逆反命题

证明方法：

  • 直接证明法：正向一步步推演

  • 对位证明法：又称逆否命题法，即证明原命题的逆否命题正确，则原命题正确

    “我妈是女人” ——> “不是女人就不是我妈”

  • 反证法：假设一个“错误”命题正确，推出矛盾，证明其错误

  • 数学归纳法：第一项+递推公式，假设第n项命题成立，证明n+1项满足命题，那么可得任一项命题成立

# 1.2、记法

元素属于集合，集合包含集合

集合：{x| x>5, x∈R}

集合X到集合Y的函数：f: X —> Y

赋值：:=

相等：▲=

# 2、向量空间和矩阵

# 2.1、向量与矩阵

n维向量：包含n个数的数组，如a=[a1,a2,...,an](t)

  • a(t)为向量a的转置，行列互换的结果

默认向量为列向量，竖着的数组捏

相加运算性质，各元素相加：

  • 交换性：a+b=b+a
  • 结合性：(a+b)+c=a+(b+c)
  • 存在零向量0=[0,0,...,0](t)使得a+0=0+a=a

减法，各元素相减，0-b记为-b：

  • b+(a-b)=a
  • -(-b)=b
  • -(a-b)-b-a

向量b-a是向量方程a+x=b的唯一解

乘法性质，各元素相乘

  • 分配性：a(a+b)=aa+bb
  • 结合性：abc=a(bc)
  • 乘1（标量1）得原数：1a=a
  • 乘0（零向量）得0：a0=0
  • 乘标量-1得：(-1)a=-a

向量集b的线性无关：a1b1+a2b2+...+anbn = 0

  • 其中a为系数，b为向量元素，当ai全为0时，我们说向量集b是线性无关的
  • 若有任一ai不为0，则b是线性相关的
  • 所有包含0向量的集合都是线性相关的
  • 若b0=a1b1+a2b2+...anbn，我们称向量b0为b1,b2,...,bn的线性组合

命题：当向量集存在一个向量是其他向量的线性组合，那么该向量集是线性相关的

命题：当向量组{a1,a2,...,an}是V的一组基，那么对于任意V中元素b，都可以用c1a1+c2a2+...+cnan唯一表示

# 2.2、矩阵的秩

矩阵的秩：矩阵A中线性无关列的最大数目，记为rankA，即为A的基的维数

运算性质：在以下运算中不改变矩阵的秩

  • 矩阵某些列乘以非零标量
  • 矩阵内部交换次序
  • 矩阵加入一列，该列是其他列的线性组合

当矩阵行列数一样，称为仿真，对应一个行列式detA或|A|

行列式运算：

  • 任一行/列为零向量，矩阵结果为0
  • 矩阵的标准基e是一个正对角线全为1，剩余位全为0的矩阵，值为1
  • 各行/列可以等比例缩小/增大，将因数提出矩阵，不影响行列式值
  • 交换行列式行/列次序，行列式符号将发生改变（交换一次变一次）

命题：若一个mxn的矩阵A具有非零的n阶子式（方阵），那么A的各列是线性无关的，即rankA=n

——> 矩阵的非零子式都是线性无关的

所以当矩阵的子式多加任意一行/列变成零子式，那么就说明原子式为矩阵的最大子式，矩阵的秩等于其非零子式的最高阶数

# 2.3、线性方程组

包含n个未知量的m个方程，共有mxn个系数，使用矩阵表示：Ax=b

  • A：mxn的系数矩阵
  • x：未知量，共n个
  • b：等式右侧（每个方程一个），共m个

增广矩阵：[A, b]

  • 即加上等式右侧值为新列

定理：方程组Ax=b有解当且仅当rankA=rank[A,b]

原方程组：

  • a11*x1+a12*x2+a1m*xm=b1
  • a21*x1+a22*x2+a2m*xm=b2
  • ......
  • an1*x1+an2*x2+anm*xm=bn

定理：当系数矩阵的秩为m（共有n列），我们可以先赋予线性相关的(n-m)列的未知数x任意值

设B=[a1,a2,...,am]

则有B[x1,x2,...,xm](t)=[b-dm+1*am+1-...-dn*an]

得[x1,x2,...,xm]=B(-1)[b-dm+1*am+1-...-dn*an]

绝对值

x、y向量的欧式内积：

for(int i = 1; i <= n; i++){
	<x, y> += x[i]y[i]
}

内积性质：

  • 非负性：<x, x>=0当且仅当x=0
  • 对称性：<x,y>=<y,x>
  • 可加性：<x+y, z>=<x,z>+<y,z>
  • 齐次性：<rx,y>=r<x,y>

|<x,y>| <= ||x||*||y||

两向量的内积小于等于两向量欧式范数的乘积

欧式范数的性质

  • 非负性：||x|| >= 0
  • 齐次性：||rx|| = r||x||
  • 三角不等式：||x+y|| <= ||x|| + ||y||

# 3.2、特征值和特征向量

对于方阵A，若|λI-A|=0，那么我们称λ为A的特征值，其中I为nxn的单位矩阵

|A-λI|=0即有n次方程成立：

  • λ(n)+a(n-1)λ(n-1)+...+a(1)λ+a(0) = 0

该多项式（等式左侧）被称为矩阵A的特征多项式，该方程称为特征方程，特征值λ便是特征方程的解（可有多个）

特征向量v满足

  • (A-λI)*v = 0

这里是矩阵乘法，等式右侧为对应维数的零向量

定理：当λ存在n个时，必存在对应不同的n个特征向量v

对称矩阵：自身和其转置相等

定理：实对称矩阵的所有特征值都是实数

定理：对于任意nxn实对称矩阵，其n个特征向量都是相互正交的

向量相互正交：内积为0

# 3.3、正交投影

  • v是Rn的子空间，相对加法和乘积是封闭

  • v的正交补v(⊥)，各分向量相互垂直

  • v(⊥)也是Rn的子空间

  • Rn=V⊕V(⊥)

    值和：对于任意Z向量∈v1+v2，有且仅有一种分解方法，那么Z∈V1⊕V2

    不同于+

    这就意味着对于任意向量x∈Rn，只能有一种正交分解方式

正交投影算子P：Px∈V且x-Px属于V(⊥)，线性变换P为正交投影算子

像空间R(A) := {Ax|x∈Rn} = span{a1,a2,...,an}

  • 矩阵A的像空间
  • x是一个列向量

零空间N(A) := {x|ax=0,x∈Rn}

  • A的零空间
  • 对应的也是一个列向量，特征方程通解
  • 零向量一定属于N(A)

定理：对于任意矩阵A，总有R(A)(⊥）=N(A(⊥))

  • 矩阵A像空间的正交补总等于矩阵A的正交补的零空间
  • Rn = N(A)⊕R(A(⊥))

定理：矩阵P是子空间V=R(P)上的正交投影算子，当且仅当 P^2 = P = P(⊥)
